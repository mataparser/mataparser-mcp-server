# Mataparser MCP Server
A Model Context Protocol (MCP) server implementation that integrates [Mataparser](https://mataparser.cloud) for document parsing capabilities.

## Features
- Parsing document or images into structured JSON
- Get API request log detail

## Installation
Please read our [documentation](https://mataparser.cloud/docs/mcp-server) to install Mataparser MCP Server.
